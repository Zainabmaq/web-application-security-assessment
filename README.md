# Web Application Security Assessment

## Overview
This project is a 3-week security assessment of OWASP NodeGoat and DVWA (Damn Vulnerable Web Application).

## Week 1 — Vulnerability Assessment
- XSS (Cross-Site Scripting) tested on DVWA
- SQL Injection tested on DVWA
- Weak password hashing (MD5) identified

## Week 2 — Security Fixes
- Input validation added using validator.js
- Password hashing implemented using bcrypt
- Token authentication added using JWT
- Secure HTTP headers added using Helmet.js

## Week 3 — Advanced Security
- Penetration testing using Nmap
- Security logging implemented using Winston
- Final security checklist created

## Tools Used
- DVWA, OWASP NodeGoat, OWASP ZAP
- Nmap, Winston, Helmet.js, bcrypt, JW
